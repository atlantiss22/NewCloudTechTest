# NewCloudTechTest
Автотесты по заданию:

Реализовать автоматические тесты.
- Зайти на сайт hh.ru, осуществить проверку, что при поиске по компаниям и словам "новые облачные", находятся "Новые Облачные Технологии"
- На странице компании "Новые облачные технологии" на сайте hh.ru проверить количество вакансий в текущем регионе
- На странице компании "Новые облачные технологии" на сайте hh.ru в текущем регионе есть вакансия "QA Automation Engineer"

Все три теста описаны в классе NewCloudTechTest.
Автотесты реализованы с использованием паттерна PageObject + Steps, могут запускаться не зависимо друг от друга.
Отчет по тесту можно получить с помощью команды "mvn allure:report", предварительно получив результаты проверки "mvn clean test".
